# Resale Cars ETL Project (SQL Server)

## Overview
This project demonstrates an end-to-end **manual ETL pipeline** built using **SQL Server and Excel**.  
Raw resale car data is extracted from CSV files, transformed using SQL, and loaded into analytical tables to support business reporting.

## ETL Process
- Extracted CSV files
- Loaded into SQL Server staging tables
- Built dimension and fact tables
- Performed data validation
- Created Excel-based analytical reports

## Tools Used
- SQL Server
- Excel
- Power Query
- GitHub

## Key Insights
- Top selling car models
- Revenue by region
- Monthly sales trends
